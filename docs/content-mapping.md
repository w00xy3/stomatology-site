## Интеграция карт (Яндекс / Google)

### Требования

1. **Яндекс Карты** — приоритет для российской аудитории
   - API: `@yandex/ymaps3` (Yandex Maps API v3)
   - 3 маркера для филиалов
   - Balloon с адресом, телефоном, графиком при клике
   - Цвет маркеров — брендовый

2. **Google Maps** — запасной вариант
   - API: `@vis.gl/react-google-maps`
   - Аналогичные маркеры

### Координаты филиалов (примерные)

| Филиал | Адрес | Координаты |
|---|---|---|
| Ставрополь 1 | ул. Орджоникидзе, 1 | `45.0433, 41.9682` |
| Ставрополь 2 | ул. Серова, 470/2 | `45.0547, 41.9730` |
| Москва | ул. Адмирала Лазарева, 52-3 | `55.5483, 37.5421` |

> **Важно:** Точные координаты необходимо уточнить через геокодирование по адресам.

### Реализация

```tsx
// components/shared/MapWidget.tsx
// Принимает: branches: Branch[]
// Рендерит: Яндекс Карту с маркерами всех филиалов
// При клике на маркер — Balloon с деталями филиала
// Переключение между филиалами через табы/чипсы
```

## Медиа-контент (изображения)

### Оптимизация

- Все изображения → **WebP** (качество 80-85)
- Использовать `next/image` для автоматической оптимизации
- Lazy loading для изображений ниже fold
- Placeholder (blur) при загрузке
