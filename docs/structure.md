# Структура сайта «Дом Стоматологии»

## Общая архитектура

```
src/
├── app/
│   ├── layout.tsx              # Root layout (Lenis, Navbar, Footer)
│   ├── page.tsx                # Главная страница
│   ├── about/
│   │   └── page.tsx            # О клинике
│   ├── services/
│   │   ├── page.tsx            # Все услуги (каталог)
│   │   └── [slug]/
│   │       └── page.tsx        # Детальная страница услуги
│   ├── doctors/
│   │   ├── page.tsx            # Все врачи
│   │   └── [slug]/
│   │       └── page.tsx        # Профиль врача
│   ├── price/
│   │   └── page.tsx            # Прайс-лист
│   ├── reviews/
│   │   └── page.tsx            # Отзывы
│   ├── promotions/
│   │   └── page.tsx            # Акции
│   ├── articles/
│   │   ├── page.tsx            # Список статей
│   │   └── [slug]/
│   │       └── page.tsx        # Детальная страница статьи
│   └── contacts/
│       └── page.tsx            # Контакты
├── components/
│   ├── layout/                 # Navbar, Footer, SmoothScroll
│   ├── ui/                     # Button, Badge, Modal, Accordion
│   ├── sections/               # Секции главной страницы
│   └── shared/                 # ServiceCard, DoctorCard, ReviewCard, MapWidget
├── data/                       # Статические данные (JSON/TS)
├── hooks/                      # useInView, useLenis
├── lib/                        # Утилиты, константы
└── styles/                     # globals.css с @theme
```

## Дизайн-принципы

- **Много «воздуха»** — generous spacing (py-20, py-32 на десктопе)
- **Типографика** — чистый sans-serif (Inter / Plus Jakarta Sans), крупные заголовки
- **Цветовая палитра:**
  - Основной: медицинский синий `#0066CC` или глубокий teal `#0D9488`
  - Акцент: тёплый `#F59E0B` (для CTA)
  - Фон: `#FAFBFC` (светло-серый) + белый
  - Текст: `#1A1A2E` (тёмный) + `#6B7280` (серый)
- **Доверие:** фото реальных врачей, лицензии, отзывы с именами
- **Конверсия:** CTA на каждой секции главной, sticky кнопка записи
